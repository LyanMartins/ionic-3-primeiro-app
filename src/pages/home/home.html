Bem vindo ao meu Aplicativo!

Esse aplicativo tem como funcionalidade aplicar conhecimento aprendidos no curso de criação de APP com IONIC 3.
O APP conta com as seguintes funcionalidades:

  • Pagina de introducao:

    • Essa pagina tem a finalidade de apresentar o aplicativo quando o usuario abrir ela pela primeira vez.

  • Pagina Home:

    • Pagina inicial que apresenta as funcionalidades do APP.

  • Pagina de Feed

    • A pagina de feed e o local onde se encontra todos os recusos usado no curso, ele se encontra uma api que tras uma lista de filmes.
    Os recursos trabalhados nessa pagina sao :

    • Consumo de API
    • Utilizaçao do componente Cards do IONIC 3
    • Utilizaçao da API InfiniteScroll do IONIC 3
    • Mais ...
  • Pagina de Configuraçao

    • Nessa pagina foi utilizado o recurso de SideMenu do IONIC 3
